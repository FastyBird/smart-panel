# Managing Your Smart Panel

Your Smart Panel is more than just a display. It’s a powerful interface that lets you control, monitor, and customize
how your smart devices interact with your environment.

This section shows you how to manage everything from adding new devices to building beautiful dashboards with tiles and
data sources.

---

### What You Can Do Here

- Add and organize smart devices
- Build pages and dashboards with drag-and-drop tiles
- Connect devices to tiles through data sources
- Use plugins to sync devices from other platforms (like Home Assistant)

Whether you’re starting from scratch or syncing existing smart devices, this section has you covered.

---

### Where This Happens

All of these actions are done through the **Admin Panel**, which runs alongside the backend and is accessible via a browser from any device on your network.

Just visit [http://smart-panel.local:3000](http://smart-panel.local:3000) or use your device’s IP address with port 3000 to access the panel.

<Callout>
	The domain `smart-panel.local` is available via mDNS when the backend is running. You can use it
	to access the Admin UI from your browser on the same network.
</Callout>

You’ll see:

- A list of your **devices**
- A collection of **pages** and **tiles**
- Tools to configure your **system**, **plugins**, and more

---

### Tip

Use the Admin Panel to:

- Experiment with new layouts
- Group sensors and controls for a room
- Easily customize the user experience without needing to code
