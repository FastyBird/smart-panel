# Projector Device

<Callout type="info">
	**Device Category**: `projector`
</Callout>

The Projector device represents a video projection display that can be powered on/off and switched between input sources.

---

## Required Channels

These channels are mandatory for the functionality of the projector:

| **Channel** | **Description** | **Multiple** | **Details** |
|-------------|-----------------|--------------|-------------|
| `television` | Controls display power, input source, and remote/navigation keys for the projector. | No | [See details](/docs/system-architecture/supported-channels/television-channel) |
| `device_information` | Provides metadata about the device, including manufacturer details, model information, and operational statuses. | No | [See details](/docs/system-architecture/supported-channels/device-information-channel) |

---

## Optional Channels

These channels are optional and provide additional features:

| **Channel** | **Description** | **Multiple** | **Details** |
|-------------|-----------------|--------------|-------------|
| `speaker` | Optional audio output control if the projector exposes speaker/volume control. | No | [See details](/docs/system-architecture/supported-channels/speaker-channel) |

---

## Use Case Scenarios

1. **Display Control**:
    - Power on/off the projector and switch between input sources. Control display settings when available.

2. **Audio Output**:
    - Control built-in speaker volume when the projector has audio output.

3. **Presentation Mode**:
    - Navigate through presentation slides using remote control keys.

---

## Developer Notes

1. The television channel is used for display control (power, input, keys).
2. Speaker channel is optional for projectors with built-in audio.
3. Consider warm-up and cool-down times for projector power cycling.
