# Television Device

<Callout type="info">
	**Device Category**: `television`
</Callout>

The Television device represents a smart TV or similar media device that can be controlled and monitored for its audio and display functionality.

---

## Required Channels

These channels are mandatory for the functionality of the television:

| **Channel** | **Description** | **Multiple** | **Details** |
|-------------|-----------------|--------------|-------------|
| `speaker` | Controls speaker devices, including volume, mute state, and audio output modes. | No | [See details](/docs/system-architecture/supported-channels/speaker-channel) |
| `television` | Provides control for TVs, including input source, brightness, and power state. | No | [See details](/docs/system-architecture/supported-channels/television-channel) |
| `device_information` | Provides metadata about the device, including manufacturer details, model information, and operational statuses. | No | [See details](/docs/system-architecture/supported-channels/device-information-channel) |

---

## Optional Channels

These channels are optional and provide additional features:

| **Channel** | **Description** | **Multiple** | **Details** |
|-------------|-----------------|--------------|-------------|
| `media_playback` | Optional playback controls when the TV exposes media player capabilities (smart TV apps). | No | [See details](/docs/system-architecture/supported-channels/media-playback-channel) |
| `media_input` | Optional input/source management when exposed as separate channel. | No | [See details](/docs/system-architecture/supported-channels/media-input-channel) |

---

## Use Case Scenarios

1. **Display Control**:
    - Power on/off, adjust brightness, and switch input sources using the television channel.

2. **Audio Control**:
    - Adjust volume and mute state using the speaker channel.

3. **Media Playback**:
    - Control built-in media apps with optional media_playback channel.

---

## Developer Notes

1. Speaker and television channels are required for core functionality.
2. Media playback and input channels support smart TV features.
3. Remote control keys are available through the television channel.
