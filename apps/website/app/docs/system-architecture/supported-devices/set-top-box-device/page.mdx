# Set-top Box Device

<Callout type="info">
	**Device Category**: `set_top_box`
</Callout>

The Set-top Box device represents a TV-connected media device (streaming box, IPTV box, or HDMI stick) providing playback control and optional source selection.

---

## Required Channels

These channels are mandatory for the functionality of the set-top box:

| **Channel** | **Description** | **Multiple** | **Details** |
|-------------|-----------------|--------------|-------------|
| `media_playback` | Provides playback controls and metadata for the set-top box. | No | [See details](/docs/system-architecture/supported-channels/media-playback-channel) |
| `device_information` | Provides metadata about the device, including manufacturer details, model information, and operational statuses. | No | [See details](/docs/system-architecture/supported-channels/device-information-channel) |

---

## Optional Channels

These channels are optional and provide additional features:

| **Channel** | **Description** | **Multiple** | **Details** |
|-------------|-----------------|--------------|-------------|
| `switcher` | Optional power control for set-top boxes (hard power / standby) when supported by the integration. | No | [See details](/docs/system-architecture/supported-channels/switcher-channel) |
| `media_input` | Optional source selection when exposed by the device (apps/inputs). | No | [See details](/docs/system-architecture/supported-channels/media-input-channel) |
| `speaker` | Optional volume control if the device exposes audio output controls. | No | [See details](/docs/system-architecture/supported-channels/speaker-channel) |

---

## Use Case Scenarios

1. **Media Streaming**:
    - Control playback of streaming content with play, pause, and navigation. View current media information and progress.

2. **App Navigation**:
    - Switch between streaming apps or input sources when available.

3. **Audio Output**:
    - Control volume when the device supports audio output.

---

## Developer Notes

1. The media_playback channel is required for playback control.
2. Switcher provides optional power control.
3. Speaker channel is optional for devices with audio output.
