# Heater Channel

<Callout type="info">
	**Channel Category**: `heater`
</Callout>

Manages heating devices, allowing control of temperature settings, modes, and valve position.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `on` | Bool | - | - | RW (Read-Write) |
| `temperature` | Float | 0–100 | °C | RW (Read-Write) |
| `status` | Bool | - | - | RO (Read-Only) |

---

### Property Details

#### `on`

Power state of the device (on/off)

#### `temperature`

Temperature reading or setpoint

#### `status`

Current operational status

## Optional Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `position` | Uchar | 0–100 | % | RW (Read-Write) |

---

### Property Details

#### `position`

Heater valve position - integrator can choose between percentage or discrete states

This property supports multiple data type variants:

- **percentage**: Uchar (0–100) in % - Position as percentage (0-100%)
- **state**: Enum (`closed`, `quarter`, `half`, `three_quarter`, `open`) - Position as discrete states
