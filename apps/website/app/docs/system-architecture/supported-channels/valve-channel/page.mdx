# Valve Channel

<Callout type="info">
	**Channel Category**: `valve`
</Callout>

Controls fluid or gas valves, managing open/close states, duration, and flow modes.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `on` | Bool | - | - | RW (Read-Write) |
| `type` | Enum | `generic`, `irrigation`, `shower_head`, `water_faucet` | - | RW (Read-Write) |

---

### Property Details

#### `on`

Power state of the device (on/off)

#### `type`

## Optional Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `duration` | Ushort | 0–3600 | s | RW (Read-Write) |
| `remaining` | Ushort | 0–3600 | s | RO (Read-Only) |
| `mode` | Enum | `manual`, `scheduled` | - | RO (Read-Only) |
| `position` | Uchar | 0–100 | % | RW (Read-Write) |
| `fault` | Uchar | 0–255 | - | RO (Read-Only) |

---

### Property Details

#### `duration`

#### `remaining`

#### `mode`

Operating mode selection

#### `position`

Valve position - integrator can choose between percentage or discrete states

This property supports multiple data type variants:

- **percentage**: Uchar (0–100) in % - Position as percentage (0-100%)
- **state**: Enum (`closed`, `quarter`, `half`, `three_quarter`, `open`) - Position as discrete states

#### `fault`

Fault code or diagnostic indicator (0 = no fault)
