# Illuminance Channel

<Callout type="info">
	**Channel Category**: `illuminance`
</Callout>

Detects ambient light levels, typically measured in lux, to monitor brightness in a space.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `illuminance` | Float | 0–100000 | lx | RO (Read-Only) |
| `level` | Enum | `bright`, `moderate`, `dusky`, `dark` | - | RO (Read-Only) |

---

### Property Details

#### `illuminance`

#### `level`

Value as discrete level (low/medium/high)

## Optional Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `active` | Bool | - | - | RO (Read-Only) |
| `fault` | Uchar | 0–255 | - | RO (Read-Only) |

---

### Property Details

#### `active`

Indicates if the channel/sensor is currently active and functioning

#### `fault`

Fault code or diagnostic indicator (0 = no fault)
