# Doorbell Channel

<Callout type="info">
	**Channel Category**: `doorbell`
</Callout>

Represents smart doorbells, providing button press detection and optional brightness control.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `event` | Enum | `single_press`, `double_press`, `long_press` | - | RO (Read-Only) |

---

### Property Details

#### `event`

## Optional Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `brightness` | Uchar | 0–100 | % | RW (Read-Write) |
| `tampered` | Bool | - | - | RO (Read-Only) |

---

### Property Details

#### `brightness`

Doorbell brightness - integrator can choose between percentage or discrete levels

This property supports multiple data type variants:

- **percentage**: Uchar (0–100) in % - Brightness as percentage (0-100%)
- **level**: Enum (`off`, `low`, `medium`, `high`, `full`) - Brightness as discrete levels

#### `tampered`

Indicates if the device has been tampered with
