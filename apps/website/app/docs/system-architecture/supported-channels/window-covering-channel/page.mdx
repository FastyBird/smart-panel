# Window Covering Channel

<Callout type="info">
	**Channel Category**: `window_covering`
</Callout>

Controls coverings like blinds, curtains, and shutters, including position and tilt adjustments.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `status` | Enum | `opened`, `closed`, `opening`, `closing`, `stopped` | - | RO (Read-Only) |
| `position` | Uchar | 0–100 | % | RW (Read-Write) |
| `type` | Enum | `curtain`, `blind`, `roller`, `outdoor_blind` | - | RO (Read-Only) |
| `command` | Enum | `open`, `close`, `stop` | - | WO (Write-Only) |

---

### Property Details

#### `status`

Current operational status

#### `position`

Window covering position - integrator can choose between percentage or discrete states

This property supports multiple data type variants:

- **percentage**: Uchar (0–100) in % - Position as percentage (0-100%)
- **state**: Enum (`closed`, `quarter`, `half`, `three_quarter`, `open`) - Position as discrete states
