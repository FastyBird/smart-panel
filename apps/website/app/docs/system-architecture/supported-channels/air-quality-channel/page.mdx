# Air Quality Channel

<Callout type="info">
	**Channel Category**: `air_quality`
</Callout>

Reports overall air quality index (AQI) and quality level classification for monitoring indoor or outdoor air conditions.

---

## Property Constraints

<Callout type="warning">
At least one property from each of the following groups must be present:
- `aqi` or `level`
</Callout>

## Optional Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `aqi` | Ushort | 0–500 | - | RO (Read-Only) |
| `level` | Enum | `excellent`, `good`, `fair`, `inferior`, `poor`, `unknown` | - | RO (Read-Only) |
| `active` | Bool | - | - | RO (Read-Only) |
| `fault` | Uchar | 0–255 | - | RO (Read-Only) |

---

### Property Details

#### `aqi`

Air Quality Index (AQI) value

#### `level`

Air quality level classification

#### `active`

Indicates if the channel/sensor is currently active and functioning

#### `fault`

Fault code or diagnostic indicator (0 = no fault)
