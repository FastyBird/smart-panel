# Microphone Channel

<Callout type="info">
	**Channel Category**: `microphone`
</Callout>

Represents audio input devices, reporting activity status and volume level.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `active` | Bool | - | - | RW (Read-Write) |

---

### Property Details

#### `active`

Indicates if the channel/sensor is currently active and functioning

## Optional Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `volume` | Uchar | 0–100 | % | RO (Read-Only) |

---

### Property Details

#### `volume`

Microphone volume/gain - integrator can choose between percentage or discrete levels

This property supports multiple data type variants:

- **percentage**: Uchar (0–100) in % - Volume as percentage (0-100%)
- **level**: Enum (`mute`, `low`, `medium`, `high`, `max`) - Volume as discrete levels
