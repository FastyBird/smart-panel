# Lock Channel

<Callout type="info">
	**Channel Category**: `lock`
</Callout>

Manages the locked/unlocked state of physical locks for doors, safes, or other secure areas.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `on` | Bool | - | - | RW (Read-Write) |
| `status` | Enum | `locked`, `unlocked` | - | RW (Read-Write) |

---

### Property Details

#### `on`

Power state of the device (on/off)

#### `status`

Current operational status
