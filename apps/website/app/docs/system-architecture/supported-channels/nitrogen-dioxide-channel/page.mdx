# Nitrogen Dioxide Channel

<Callout type="info">
	**Channel Category**: `nitrogen_dioxide`
</Callout>

Monitors the concentration of NO2 in the air, often used in environmental and industrial monitoring.

---

## Property Constraints

<Callout type="warning">
At least one property from each of the following groups must be present:
- `detected` or `concentration`
</Callout>

## Optional Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `detected` | Bool | - | - | RO (Read-Only) |
| `concentration` | Float | 0–100000 | µg/m³ | RO (Read-Only) |
| `mode` | Enum | `annual`, `1_hour` | - | RO (Read-Only) |
| `active` | Bool | - | - | RO (Read-Only) |
| `fault` | Uchar | 0–255 | - | RO (Read-Only) |
| `tampered` | Bool | - | - | RO (Read-Only) |

---

### Property Details

#### `detected`

Binary detection state (true = detected)

#### `concentration`

Measured concentration level

#### `mode`

Operating mode selection

#### `active`

Indicates if the channel/sensor is currently active and functioning

#### `fault`

Fault code or diagnostic indicator (0 = no fault)

#### `tampered`

Indicates if the device has been tampered with
