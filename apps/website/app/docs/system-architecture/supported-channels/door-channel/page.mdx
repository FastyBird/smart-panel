# Door Channel

<Callout type="info">
	**Channel Category**: `door`
</Callout>

Manages the open/close states of doors, including garage and entry doors with position tracking.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `obstruction` | Bool | - | - | RO (Read-Only) |
| `status` | Enum | `opened`, `closed`, `opening`, `closing`, `stopped` | - | RO (Read-Only) |
| `position` | Enum | `open`, `close`, `stop` | - | RW (Read-Write) |
| `type` | Enum | `door`, `garage` | - | RO (Read-Only) |

---

### Property Details

#### `obstruction`

#### `status`

Current operational status

#### `position`

Position value

#### `type`

## Optional Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|

| `fault` | Uchar | 0–255 | - | RO (Read-Only) |

### Property Details

#### `percentage`

Value as percentage (0-100%)
