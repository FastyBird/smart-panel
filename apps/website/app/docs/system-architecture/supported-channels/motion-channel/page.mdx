# Motion Channel

<Callout type="info">
	**Channel Category**: `motion`
</Callout>

Detects motion in an area to trigger alerts or automate responses.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `detected` | Bool | - | - | RO (Read-Only) |

---

### Property Details

#### `detected`

Binary detection state (true = detected)

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |

| `active` | Bool | - | - | RO (Read-Only) |
| `fault` | Uchar | 0–255 | - | RO (Read-Only) |
| `tampered` | Bool | - | - | RO (Read-Only) |

---

### Property Details

#### `distance`

#### `active`
