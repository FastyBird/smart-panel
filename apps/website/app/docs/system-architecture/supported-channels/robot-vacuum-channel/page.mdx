# Robotic Vacuum Channel

<Callout type="info">
	**Channel Category**: `robot_vacuum`
</Callout>

Controls robotic vacuum cleaners, providing commands for start, stop, docking, and mode selection.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `on` | Bool | - | - | RW (Read-Write) |
| `status` | Enum | `idle`, `cleaning`, `vacuuming`, `mopping`, `docking`, `charging` | - | RO (Read-Only) |

---

### Property Details

#### `on`

Power state of the device (on/off)

#### `status`

Current operational status
