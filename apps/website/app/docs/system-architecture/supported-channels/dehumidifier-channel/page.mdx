# Dehumidifier Channel

<Callout type="info">
	**Channel Category**: `dehumidifier`
</Callout>

Controls dehumidifier devices, including on/off state, target humidity, and operating modes.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `on` | Bool | - | - | RW (Read-Write) |
| `humidity` | Uchar | 0–100 | % | RW (Read-Write) |

---

### Property Details

#### `on`

Turn the dehumidifier on or off

#### `humidity`

Target humidity level

## Optional Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `mode` | Enum | `auto`, `manual`, `continuous`, `laundry`, `quiet` | - | RW (Read-Write) |
| `status` | Enum | `idle`, `dehumidifying`, `defrosting` | - | RO (Read-Only) |
| `water_tank_full` | Bool | - | - | RO (Read-Only) |
| `water_tank_level` | Uchar | 0–100 | % | RO (Read-Only) |
| `defrost_active` | Bool | - | - | RO (Read-Only) |
| `timer` | Ushort | 0–86400 | s | RW (Read-Write) |
| `locked` | Bool | - | - | RW (Read-Write) |
| `fault` | Uchar | 0–255 | - | RO (Read-Only) |

---

### Property Details

#### `mode`

Operating mode of the dehumidifier

#### `status`

Current operational status

#### `water_tank_full`

Water tank full indicator

#### `water_tank_level`

Water tank level as percentage

#### `defrost_active`

Defrost cycle is active

#### `timer`

Auto-off timer - integrator can choose between duration or preset intervals

This property supports multiple data type variants:

- **duration**: Ushort (0–86400) in s - Timer duration in seconds (0-86400)
- **preset**: Enum (`off`, `30m`, `1h`, `2h`, `4h`, `8h`, `12h`) - Preset timer intervals

#### `locked`

Child lock or panel lock state

#### `fault`

Fault code or diagnostic indicator (0 = no fault)
