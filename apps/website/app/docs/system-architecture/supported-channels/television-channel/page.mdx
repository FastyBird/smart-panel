# Television Channel

<Callout type="info">
	**Channel Category**: `television`
</Callout>

Provides control for TVs and displays, including power, input source, brightness, and remote keys.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `on` | Bool | - | - | RW (Read-Write) |
| `source` | Enum | `hdmi1`, `hdmi2`, `hdmi3`, `hdmi4`, `hdmi5`, `hdmi6`, `arc`, `earc`, `tv`, `cable`, `satellite`, `antenna`, `av1`, `av2`, `component`, `vga`, `dvi`, `usb`, `bluetooth`, `wifi`, `airplay`, `cast`, `dlna`, `miracast`, `app_netflix`, `app_youtube`, `app_spotify`, `app_prime_video`, `app_disney_plus`, `app_hbo_max`, `app_apple_tv`, `app_plex`, `app_kodi`, `other` | - | RW (Read-Write) |

---

### Property Details

#### `on`

Power state of the device (on/off)

#### `source`

Select input/source using either a standardized identifier or an integration-specific value.

This property supports multiple data type variants:

- **standard**: Enum (`hdmi1`, `hdmi2`, `hdmi3`, `hdmi4`, `hdmi5`, `hdmi6`, `arc`, `earc`, `tv`, `cable`, `satellite`, `antenna`, `av1`, `av2`, `component`, `vga`, `dvi`, `usb`, `bluetooth`, `wifi`, `airplay`, `cast`, `dlna`, `miracast`, `app_netflix`, `app_youtube`, `app_spotify`, `app_prime_video`, `app_disney_plus`, `app_hbo_max`, `app_apple_tv`, `app_plex`, `app_kodi`, `other`) - Standardized input/source identifier (recommended for UI).
- **custom**: String - Integration-specific input/source value (fallback).

## Optional Properties

|--------------|---------------|------------------|----------|-----------------|
| `brightness` | Uchar | 0–100 | % | RW (Read-Write) |
| `source_label` | String | - | - | RO (Read-Only) |
| `remote_key` | Enum | `rewind`, `fast_forward`, `next`, `previous`, `arrow_up`, `arrow_down`, `arrow_left`, `arrow_right`, `select`, `back`, `exit`, `info`, `play`, `pause` | - | WO (Write-Only) |

---

### Property Details

#### `brightness`

TV brightness - integrator can choose between percentage or discrete levels

This property supports multiple data type variants:

- **percentage**: Uchar (0–100) in % - Brightness as percentage (0-100%)
- **level**: Enum (`off`, `low`, `medium`, `high`, `full`) - Brightness as discrete levels

#### `source_label`

Human-readable label for the current input/source.
