# Camera Channel

<Callout type="info">
	**Channel Category**: `camera`
</Callout>

Provides video and image data streams from surveillance or monitoring cameras with PTZ and infrared controls.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `status` | Enum | `available`, `in_use`, `unavailable`, `offline`, `initializing`, `error` | - | RO (Read-Only) |
| `source` | String | - | - | RO (Read-Only) |

---

### Property Details

#### `status`

Current operational status

#### `source`

Input source selection

## Optional Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `zoom` | Float | 1–10 | - | RW (Read-Write) |
| `pan` | Float | -180–180 | ° | RW (Read-Write) |
| `tilt` | Float | -90–90 | ° | RW (Read-Write) |
| `infrared` | Bool | - | - | RW (Read-Write) |
| `fault` | Uchar | 0–255 | - | RO (Read-Only) |

---

### Property Details

#### `zoom`

#### `pan`

#### `tilt`

#### `infrared`

#### `fault`

Fault code or diagnostic indicator (0 = no fault)
