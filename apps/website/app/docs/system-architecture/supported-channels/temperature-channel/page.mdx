# Temperature Channel

<Callout type="info">
	**Channel Category**: `temperature`
</Callout>

Measures the current temperature in an environment, reported in degrees Celsius.

---

## Required Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `temperature` | Float | 0–100 | °C | RO (Read-Only) |

---

### Property Details

#### `temperature`

Temperature reading or setpoint

## Optional Properties

| **Property** | **Data Type** | **Range/Values** | **Unit** | **Permissions** |
|--------------|---------------|------------------|----------|-----------------|
| `active` | Bool | - | - | RO (Read-Only) |
| `fault` | Uchar | 0–255 | - | RO (Read-Only) |

---

### Property Details

#### `active`

Indicates if the channel/sensor is currently active and functioning

#### `fault`

Fault code or diagnostic indicator (0 = no fault)
