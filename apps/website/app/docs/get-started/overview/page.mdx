# Get Started

Curious about building your own smart home dashboard? You’re in the right place. This section will walk you through everything
you need to turn your Raspberry Pi into a fully functional Smart Panel — no deep tech knowledge required.

If you’ve ever flashed an SD card or set up a Raspberry Pi, you’re more than ready to get started.

---

## What You’ll Learn

- What you need before you start
- How to prepare your Raspberry Pi
- Choosing the right deployment scenario for your setup
- Installing using pre-built distribution packages
- What to expect on first boot

---

## What You’ll Have at the End

A working Smart Panel that boots into a sleek UI, connects to your network, and is ready to manage your smart home devices.

Smart Panel supports three deployment scenarios:

- **Server Only** — run just the backend on a central device, with display panels connecting remotely
- **Display Only** — add a display panel to an existing server on your network
- **All-in-One** — run everything on a single device (great for a standalone Raspberry Pi panel)

👉 Start with [**Before You Begin**](/docs/get-started/before-you-begin) to get everything in place.
