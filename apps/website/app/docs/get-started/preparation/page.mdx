# Preparation

Let's get your **Raspberry Pi** up and running with the operating system and the Smart Panel software. This part will
walk you through flashing the SD card and doing the first boot.

---

## Option 1: Pre-Built Smart Panel Image (Recommended)

The fastest way to get started is to flash one of the **pre-built Raspberry Pi OS images**. Everything is already
installed — the backend, admin interface, and display app — so there is no manual setup required.

### Image Variants

| Variant | What's Included | Best For |
| --- | --- | --- |
| **Server** | Backend + Admin UI | Headless server without a display |
| **All-in-One (AIO)** | Backend + Admin UI + Display App | Single device with a touchscreen |
| **Display** | Display App only | Dedicated display that connects to a remote server |

<Callout type="info">
	All images are **ARM64 only** and support Raspberry Pi 3B+, Pi 4, Pi 5, and Pi Zero 2W (64-bit).
</Callout>

<Steps>
	### Download the Image

	Go to the [GitHub Releases](https://github.com/FastyBird/smart-panel/releases) page and download the image for your
	variant:

	- `smart-panel-server-{version}-arm64.img.xz`
	- `smart-panel-aio-{version}-arm64.img.xz`
	- `smart-panel-display-{version}-arm64.img.xz`

	### Flash with Raspberry Pi Imager

	1. Open [Raspberry Pi Imager](https://www.raspberrypi.com/software/)
	2. Click **Choose OS**, scroll to the bottom and select **Use custom**
	3. Select the downloaded `.img.xz` file
	4. Choose your SD card and click **Next**, then **Yes** to flash

	### (Optional) Pre-configure WiFi and Hostname

	Before inserting the SD card, mount the boot partition and create a file named `smart-panel.conf` with any of the
	following settings:

	```ini
	WIFI_SSID=MyNetwork
	WIFI_PASSWORD=mypassword
	WIFI_COUNTRY=US
	HOSTNAME=smart-panel
	TIMEZONE=America/New_York
	```

	All fields are optional. If you skip WiFi configuration, a captive portal will handle it on first boot.

	### Insert the SD Card and Power On

	Insert the microSD card into your Raspberry Pi and power it on. On first boot the image will:

	1. Auto-expand the root partition to fill the SD card
	2. Apply WiFi/hostname/timezone from `smart-panel.conf` (if present)
	3. Compile native modules and run database migrations
	4. Start all services automatically
</Steps>

<Callout>
	No WiFi config file? No problem — a WiFi setup hotspot appears automatically (AP name `SmartPanel-XXXX`,
	password `smartpanel`). See [Captive Portal](/docs/get-started/installation/captive-portal) for details.
</Callout>

## Option 2: Vanilla Raspberry Pi OS

If you prefer to start from a stock OS and install everything yourself, use the **Raspberry Pi Imager** to flash a
vanilla Raspberry Pi OS image.

<Steps>
	### Download Raspberry Pi Imager

	You can get the Raspberry Pi Imager from the official site:

	[https://www.raspberrypi.com/software](https://www.raspberrypi.com/software)

	It's available for Windows, macOS, and Linux.

	### Insert the SD Card

	Insert a microSD card (at least 8GB recommended) into your computer's card reader.

	### Choose Operating System

	In Raspberry Pi Imager:

	1. Click Choose OS
	2. Select Raspberry Pi OS Lite (64-bit) (This is a lightweight, headless version perfect for embedded panels)

	<p align="center">
		<img src="/images/imager-select-os.png" alt="Choosing the OS in Raspberry Pi Imager" />
	</p>

	### Configure Advanced Options

	Click the **Next** and then **Edit settings**.

	Set the following:

	On the **General** tab:

	- Set username and password (e.g. pi / raspberry)
	- Configure Wi-Fi (SSID, password, country)
	- Set hostname (e.g. smart-panel.local)

	On the **Services** tab:

	- Enable SSH

	<p align="center">
		<img src="/images/imager-settings.png" alt="Raspberry Pi Imager settings" />
	</p>

	<Callout>
		These options are saved per session. You can optionally make them default for future flashes.
	</Callout>

	When finished, click **Yes** to apply the settings.

	<p align="center">
		<img src="/images/imager-confirm-settings.png" alt="Raspberry Pi Imager confirming settings" />
	</p>

	### Flash the OS

	Click **Yes** to begin flashing the image. This may take a few minutes.

	<p align="center">
		<img src="/images/imager-write-sd-card.png" alt="Raspberry Pi Imager writing to SD card" />
	</p>

	Once done, remove the SD card safely and insert it into your Raspberry Pi.
</Steps>

## Advanced: Using Balena Etcher

If you're a more advanced user or prefer full control, you can flash the image manually using [balenaEtcher](https://etcher.io/)

<Steps>
	### Download the OS

	We recommend using [Raspberry Pi OS Lite](https://www.raspberrypi.com/software/operating-systems/), since it's
	lightweight and ideal for embedded use.

	### Flash the SD Card

	1. Insert the microSD card into your computer.
	2. Use Etcher to write the OS image.
	3. Once flashing is done, don't remove the card just yet!

	### Enable Wi-Fi

	Create a file named `wpa_supplicant.conf` with your Wi-Fi credentials:

	```conf
	country=CZ
	ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
	update_config=1

	network={
		ssid="YourSSID"
		psk="YourPassword"
	}
	```

	### Enable SSH

	Create an **empty file** called `ssh` (with no extension) to enable SSH.

	### Finalize

	Save both files in the boot partition before inserting the SD card into your Pi.
</Steps>

## First Boot

1. Insert the microSD into the Pi.
2. Power it on.
3. Wait a minute or two for the initial setup.

**You can connect to your Pi via SSH using:**

```bash
ssh pi@smart-panel.local
```

<Callout type="info">
	If you set a custom hostname during flashing, use that instead. The default Raspberry Pi hostname is `raspberrypi.local`.
	If you followed the recommended settings above, use `smart-panel.local`.
</Callout>
