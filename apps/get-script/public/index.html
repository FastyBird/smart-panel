Smart Panel

Install on any Linux device with one command.

Server (Backend + Admin UI)

click to copy curl -fsSL https://get.smart-panel.fastybird.com | sudo bash

Display Panel

click to copy curl -fsSL https://get.smart-panel.fastybird.com/display | sudo bash \ -s -- --backend http://YOUR_SERVER_IP:3000

Replace YOUR_SERVER_IP with your server's IP address.

All-in-One (Server + Display)

click to copy curl -fsSL https://get.smart-panel.fastybird.com | sudo bash && \ curl -fsSL https://get.smart-panel.fastybird.com/display | sudo bash \ -s -- --backend http://localhost:3000 --kiosk
