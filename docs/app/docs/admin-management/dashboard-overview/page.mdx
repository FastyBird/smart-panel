# Customize Dashboard

Design your Smart Panel exactly the way you want it. The dashboard lets you organize and visualize your smart devices
using interactive pages, tiles, and cards — with full flexibility.

<Callout type="info">
	Your dashboard is built using a modular layout of pages, tiles, and data sources. You can mix and match layouts, organize
	devices, and tailor views for different screens.
</Callout>

## What This Section Covers

- How to create pages for your panel UI
- Add and arrange tiles and cards
- Link data from devices to UI components
- Use data sources to visualize device states

After completing this section, you’ll have a fully interactive Smart Panel layout showing real-time device data, with pages
and components tailored to your needs.
