Work on a task from the tasks/ directory.

Usage: /task <task-id>

Example: /task FEATURE-PLUGIN-WLED

Steps:
1. Read the task file from `tasks/features/<task-id>.md` or `tasks/technical/<task-id>.md`
2. Check if the task is already done (Status: done)
3. If not done, review the acceptance criteria
4. Propose an implementation plan before making changes
5. Follow the AI instructions in the task file
6. Update the task status when complete
